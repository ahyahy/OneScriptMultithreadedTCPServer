Кодировка.ASCII (Encoding.ASCII) Свойство

Использование

Только чтение.

Значение

Тип: Кодировка (Encoding).

Возвращает кодировку для набора символов ASCII (7-разрядных).

Примечание

Символы ASCII ограничены самыми низкими 128 символами Unicode, начиная с U + 0000 до U + 007f.

Пример

Кодировка3 = МС.Кодировка().ASCII;
