Кодировка.ПолучитьКодировку (Encoding.GetEncoding) Метод

Синтаксис

ПолучитьКодировку(<КодоваяСтраница>)

Параметры

КодоваяСтраница (обязательный)
Тип: Число.
Идентификатор кодовой страницы получаемой кодировки.

Возвращаемое значение

Тип: Кодировка (Encoding).

Кодировка, связанная с заданной кодовой страницей.

Описание

Возвращает кодировку для указанной кодовой страницы.

Примечание

Пример

Кодировка = МС.Кодировка().ПолучитьКодировку(37);
Сообщить("" + Кодировка.ИмяWeb + "	" + Кодировка.ИмяКодировки);
