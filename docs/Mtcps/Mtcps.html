Библиотека MultithreadedTCPServer Классы

Классы

TCPКлиент (TCPClient) Предоставляет клиентские подключения для сетевых служб протокола TCP.
АргументыСобытия (EventArgs) Представляет класс, содержащий данные событий.
Действие (Action) Задает обработчик события.
Кодировка (Encoding) Представляет кодировку символов.
МногопоточныйTCPСервер (MultithreadedTCPServer) Представляет многопоточный TCP сервер, который используется для приема клиентских подключений и взаимодействия с ними.
МногопоточныйTCPСерверДляОдноСкрипта (OneScriptMultithreadedTCPServer) Объект, содержащий свойства и методы для управления многопоточным TCP сервером и TCP клиентом, а так же необходимые для этого объекты.
ПотокСети (NetworkStream) Обеспечивает базовый поток данных для доступа к сети.
