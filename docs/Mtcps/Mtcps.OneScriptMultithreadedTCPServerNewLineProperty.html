МногопоточныйTCPСерверДляОдноСкрипта.НоваяСтрока (OneScriptMultithreadedTCPServer.NewLine) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает строку, обозначающую в данной среде начало новой строки.

Примечание

Пример

Текст = "Апрель, апрель!" + МС.НоваяСтрока + 
"На дворе звенит капель." + МС.НоваяСтрока + 
"По полям бегут ручьи," + МС.НоваяСтрока + 
"На дорогах лужи." + МС.НоваяСтрока + 
"Скоро выйдут муравьи" + МС.НоваяСтрока + 
"После зимней стужи.";
Полный пример кода
