Кодировка.ИмяТела (Encoding.BodyName) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает имя текущей кодировки, которое может использоваться с тегами текста сообщения почтового агента.

Примечание

Например, если эта кодировка эквивалентна UTF8Encoding, это свойство возвращает "utf-8". Если кодировка не может быть использована, значением свойства является пустая строка ("").

Пример

Кодировка3 = МС.Кодировка().UTF8;
Сообщить("Кодировка3.ИмяТела = " + Кодировка3.ИмяТела);
