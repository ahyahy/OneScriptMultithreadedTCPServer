Кодировка.ПоУмолчанию (Encoding.ByDefault) Свойство

Использование

Только чтение.

Значение

Тип: Кодировка (Encoding).

Возвращает кодировку для текущей кодовой страницы ANSI операционной системы.

Примечание

Пример

Кодировка3 = МС.Кодировка().ПоУмолчанию;
Сообщить("Кодировка3.КодоваяСтраница = " + Кодировка3.КодоваяСтраница);
