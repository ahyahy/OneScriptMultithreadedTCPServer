ПотокСети (NetworkStream) Свойства

Свойства

ВозможностьЗаписи (CanWrite) Возвращает значение, указывающее, поддерживает ли объект запись. Значение Истина, если данные могут быть записаны в поток, в противном случае - Ложь. Значение по умолчанию: Истина.
ВозможностьЧтения (CanRead) Возвращает значение, указывающее, поддерживает ли объект чтение. Значение Истина, если данные могут быть прочитаны из потока, в противном случае - Ложь. Значение по умолчанию: Истина.
ДанныеДоступны (DataAvailable) Возвращает значение, указывающее, имеются ли в объекте данные, доступные для чтения. Значение Истина, если содержащиеся в потоке данные могут быть прочитаны, в противном случае - Ложь. Значение по умолчанию: Истина.
