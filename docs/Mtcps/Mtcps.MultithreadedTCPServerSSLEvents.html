События

ПриОтключенииКлиента (ClientDisconnected) Возвращает или задает код для выполнения при отключении клиента.
ПриОшибкеСервера (ErrorServer) Возвращает или задает код для выполнения при возникновении ошибки или предупреждения на сервере.
ПриПодключенииКлиента (ClientConnected) Возвращает или задает код для выполнения при подключении клиента.
СерверПолучилДанные (ServerReceived) Возвращает или задает код для выполнения при получении сервером данных.
