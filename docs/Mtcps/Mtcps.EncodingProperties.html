Кодировка (Encoding) Свойства

Свойства

ASCII (ASCII) Возвращает кодировку для набора символов ASCII (7-разрядных).
UTF7 (UTF7) Возвращает кодировку для формата UTF-7.
UTF8 (UTF8) Возвращает кодировку для формата UTF-8.
ИмяWeb (WebName) Возвращает для текущей кодировки имя, зарегистрированное в IANA (Internet Assigned Numbers Authority).
ИмяЗаголовка (HeaderName) Возвращает имя текущей кодировки, которое может использоваться с тегами заголовка сообщения почтового агента.
ИмяКодировки (EncodingName) Возвращает описание текущей кодировки, которое может быть прочитано пользователем.
ИмяТела (BodyName) Возвращает имя текущей кодировки, которое может использоваться с тегами текста сообщения почтового агента.
КодоваяСтраница (WindowsCodePage) Возвращает кодовую страницу операционной системы Windows, наиболее точно соответствующую текущей кодировке.
ОбратнаяUTF16 (BigEndianUnicode) Возвращает кодировку для формата UTF-16 с обратным порядком байтов.
ПоУмолчанию (ByDefault) Возвращает кодировку для текущей кодовой страницы ANSI операционной системы.
Юникод (Unicode) Возвращает кодировку для формата UTF-16 с прямым порядком байтов.
